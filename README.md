📦 Campus Lost & Found

A simple full-stack web application that helps students report and find lost items on campus. Users can view listings, filter by status (lost/found), and add new items easily.

🚀 Features
📋 View all lost & found items
🔍 Filter items by:
Lost
Found
➕ Add new items with:
Title
Description
Location
Status
Image URL
🖼️ Clean card-based UI
🌐 Backend API using Flask
☁️ Database powered by Supabase
🛠️ Tech Stack
Frontend
    HTML
    CSS
    JavaScript (Vanilla JS)
Backend
    Flask
Database
    Supabase (PostgreSQL)
